# Formatting Selectors

This file defines the named formatting selectors available in the DohDocs
"F" formatting ribbon. Each selector maps to a CSS class (`fmt-<id>`) defined
in `src/styles/formatting-selectors.css`.

| id      | label   | description                          |
|---------|---------|--------------------------------------|
| p1      | P1      | Red highlight (priority 1)           |
| p2      | P2      | Yellow highlight (priority 2)        |
| p3      | P3      | Blue highlight (priority 3)          |
| comment | Comment | Italic, wrapped in quotation marks   |
| math    | Math    | Inline text-based calculator         |
